# Language System — Meri Moreira Correspondente

> Guia completo de tom, vocabulário, estrutura de mensagem e exemplos de copy para todos os canais.

---

## 1. Essência da Voz

A Meri Moreira fala como uma **especialista de confiança que você já conhece** — não como um banco, não como um vendedor, não como um robô.

**A marca fala como Meri:** uma profissional experiente, curitibana, que entende cada detalhe do processo e sente alegria genuína quando o cliente conquista a casa própria.

---

## 2. Os 4 Pilares de Linguagem

| Pilar | Descrição | Anti-padrão |
|-------|-----------|-------------|
| **Direta** | Vai ao ponto. Sem enrolação, sem jargão desnecessário | ❌ "Mediante análise criteriosa das condicionantes..." |
| **Acolhedora** | Reconhece que é uma decisão enorme. Não pressiona | ❌ "ÚLTIMA CHANCE! Não perca essa oportunidade!" |
| **Especialista** | Transmite autoridade com simplicidade. Explica sem complicar | ❌ "Alienação fiduciária do imóvel via SFH/SFI" |
| **Encorajadora** | Remove o medo. Estimula quem acha que "não vai conseguir" | ❌ "Depende muito do seu perfil e histórico..." |

## 3. Tom por Canal

### WhatsApp (atendimento direto)
- **Tom:** Pessoal, próximo, eficiente
- **Pronome:** "você" sempre. Nunca "o senhor/a senhora"
- **Cumprimento:** "Oi, [nome]!" ou "Olá! Tudo bem?"
- **Extensão:** Mensagens curtas. Máximo 3–4 linhas por balão
- **Emoji:** Com moderação — 1 a 2 por mensagem, nunca em excesso
- **Encerramento:** Sempre com próximo passo claro

### Instagram (posts e legendas)
- **Tom:** Leve, informativo, com personalidade
- **Abertura:** Frase de impacto ou pergunta que gera identificação
- **Corpo:** Informação útil, explicada de forma simples
- **CTA:** Sempre presente — "Chama no WhatsApp 👇" ou "Manda uma mensagem"
- **Hashtags:** 5–10, mix de nicho (#financiamentoimobiliario) e local (#curitiba #imovelcuritiba)

### Stories
- **Tom:** Conversa direta, como se falasse para um amigo
- **Extensão:** 1 ideia por story. Máximo 2 frases de texto
- **CTA:** Sempre "Arrasta pra cima" ou "Responde aqui"

### Site / Landing Page
- **Tom:** Confiante, claro, com prova social
- **Estrutura:** Benefício → Prova → CTA
- **Linguagem:** Sem jargão técnico nas áreas visíveis. Técnico apenas onde esperado (FAQ)

### Google Meu Negócio
- **Tom:** Formal, mas acessível
- **Foco:** Credibilidade, localização, especialização
- **Palavras-chave:** "Curitiba", "Caixa", "FGTS", "Minha Casa Minha Vida"

## 4. Vocabulário da Marca

### Usar com frequência ✅

| Palavra/Expressão | Por quê funciona |
|-------------------|-----------------|
| Casa própria | Emocional, objetivo do cliente |
| Lar | Mais afetivo que "imóvel" |
| Conquistar / realizar | Linguagem de vitória, não de compra |
| Viabilizar | Palavra-chave do negócio — diferencia a Meri |
| Simples / sem complicação | Remove o medo da burocracia |
| A gente cuida de tudo | Promessa central |
| +15 anos | Prova de credibilidade imediata |
| Centenas de aprovações | Prova social específica |
| Curitiba / Bacacheri / região | Conexão local, autenticidade |
| FGTS | Ponto de entrada emocional |
| Minha Casa Minha Vida | Programa que define o público |
| Sem custo para você | Remove objeção de preço |

### Evitar ❌

| Palavra/Expressão | Por quê evitar |
|-------------------|---------------|
| Promoção / Oferta imperdível | Tom de vendas agressivo |
| Alienação fiduciária (sem explicar) | Jargão que assusta |
| Amortização (sem explicar) | Idem |
| Último dia / Vagas limitadas | Pressão artificial |
| O senhor / A senhora | Distancia, envelhece |
| Processo burocrático | Reforça o medo |
| Não há garantias | Insegurança desnecessária |
| Clicar aqui | Vago demais |

## 7. Explicações Simplificadas de Termos Técnicos

Use estas versões ao escrever para o público geral:

| Termo técnico | Como explicar para o cliente |
|--------------|------------------------------|
| Correspondente Credenciado | "Somos parceiros autorizados da Caixa — cuidamos de todo o processo do financiamento por você" |
| SFH / SFI | Não usar. Dizer apenas "financiamento pela Caixa" |
| Alienação fiduciária | "O imóvel fica registrado no seu nome, mas como garantia do banco até você pagar" |
| Amortização | "Reduzir o saldo da dívida" |
| LTV | Não usar. Dizer "percentual financiado" |
| ITBI | "Imposto da prefeitura para transferir o imóvel para o seu nome" |
| Matrícula | "Documento oficial do imóvel no cartório" |
| Laudo de engenharia | "Avaliação técnica que o banco faz no imóvel antes de aprovar" |

## 8. Checklist de Revisão de Texto

Antes de publicar ou enviar qualquer texto, verificar:

- [ ] Tom está direto mas acolhedor?
- [ ] Tem CTA claro com contato ou link do WhatsApp?
- [ ] Usa vocabulário da marca (sem jargão técnico solto)?
- [ ] Remove objeções ou medos sem criar ansiedade?
- [ ] Menciona localização (Curitiba) quando relevante?
- [ ] Está no tamanho certo para o canal?
- [ ] Tem prova social ou credencial (+15 anos, centenas de aprovações)?
- [ ] Não usa linguagem agressiva de vendas?
