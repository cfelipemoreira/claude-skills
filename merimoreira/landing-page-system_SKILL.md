# Landing Page System — Meri Moreira Correspondente

> Arquitetura completa de conversão para o site/landing page principal.
> Toda página criada deve ter como objetivo único: **iniciar uma conversa no WhatsApp**.

---

## 1. Princípio Central de Conversão

O público da Meri tem **medo de não conseguir** o financiamento. A landing page não vende um serviço — ela **remove esse medo** e convida para dar o primeiro passo.

Cada seção deve responder a uma objeção implícita:

| Seção | Objeção que resolve |
|-------|---------------------|
| Hero | "Será que isso é pra mim?" |
| Como funciona | "É complicado demais" |
| Serviços | "Não sei o que vocês fazem exatamente" |
| Diferenciais | "Por que a Meri e não ir direto no banco?" |
| Prova Social | "Será que funciona de verdade?" |
| FAQ | "Tenho dúvidas específicas" |
| CTA Final | "Ok, mas qual é o próximo passo?" |

## 2. Estrutura Completa da Landing Page

### SEÇÃO 1 — HERO (Above the fold)

**Objetivo:** Comunicar o valor em menos de 5 segundos. CTA imediato.

**Elementos obrigatórios:**
- Logo / nome da marca
- Headline principal (H1)
- Subheadline (H2)
- Botão CTA primário → WhatsApp
- Credencial de confiança (sutil, abaixo do CTA)
- Imagem de fundo ou foto (família/casa/chave)

**Copy sugerido:**

```
HEADLINE (H1):
"Sua casa própria começa aqui."

SUBHEADLINE (H2):
"Cuidamos de todo o processo de financiamento Caixa — 
do FGTS ao Minha Casa Minha Vida. Mais de 15 anos 
realizando sonhos em Curitiba."

CTA:
[Falar no WhatsApp agora]

CREDENCIAL:
🏠 Correspondente Credenciada Caixa · +15 anos · Curitiba e RMC
```

**Design:**
- Fundo: branco ou azul escuro (`#1C60AB`) com foto em overlay
- Tipografia: Neue Gstaad para H1, Montserrat para H2
- Botão: verde WhatsApp (`#25D366`) com texto branco — exceção visual justificada pela força do canal
- Espaço generoso acima do fold

## 3. Regras de Conversão

- **CTA principal = WhatsApp sempre.** Nunca formulário como primário.
- **Botão WhatsApp:** Deve aparecer em pelo menos 4 pontos da página (Hero, Como funciona, CTA final, Fixo no mobile)
- **Botão fixo mobile:** WhatsApp flutuante fixo no canto inferior direito em toda a página
- **Velocidade:** Página deve carregar em < 3s. Imagens otimizadas.
- **Mobile-first:** +70% do público acessa pelo celular. Layout deve ser perfeito no mobile primeiro.
- **Formulários:** Se usar formulário, máximo 3 campos (Nome, Telefone, Qual serviço). Nada mais.
