# Nex — Base Knowledge

## O que é o Nex

O Nex é um **ecossistema de espaços de trabalho, comunidade e experiências** voltado para o futuro do trabalho.

Não é apenas coworking. É uma plataforma de conexões profissionais, criatividade e crescimento que integra:
- Trabalho
- Networking
- Cultura
- Experiências

> O Nex transforma o ambiente de trabalho em um ativo estratégico de relacionamento, geração de negócios e desenvolvimento profissional.

---

## Posicionamento Central

**O Nex é um hub contemporâneo de trabalho e conexões**, que combina infraestrutura de alto nível com comunidade ativa.

### O Nex NÃO é:
- Apenas coworking
- Um produto que compete por preço
- Um espaço que vende só metro quadrado

### O Nex VENDE:
- Ambiente
- Experiência
- Relacionamento
- Pertencimento
- Acesso

## Revenue Flywheel

Qualquer produto pode ser porta de entrada para outro:

```
Day Pass → Plano Mensal
Escritório Virtual → Uso Físico
Eventos → Leads Corporativos
Nex House → Upsell de Privativo
Sala de Reunião → Captação de Resident
```

Resultado: maior LTV, maior frequência, maior ticket médio, menor CAC.

## Regras Gerais de Aplicação

Antes de gerar qualquer conteúdo para o Nex:

1. **Identifique o produto ou contexto** — cada produto tem lógica própria
2. **Consulte o design system** — cores, tipografia e layout seguem regras fixas
3. **Aplique o tom correto** — contemporâneo, profissional, humano, limpo
4. **Não misture narrativas** — cada produto tem seu argumento central
5. **Foque em utilidade** — clareza sempre supera decoração verbal
